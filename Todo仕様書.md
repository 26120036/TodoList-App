# TODOリスト管理システム仕様書（Markdown）

## 1. 概要

### 課題名

TODOリスト管理システム

### 目的

コンソール上でTODO（やること）を管理できるプログラムを作成する。

登録したタスクはファイルへ保存し、次回起動時に読み込めるようにする。

---

# 2. 学習目標

* クラス設計
* ArrayListの利用
* メソッドの作成
* while文・switch文
* Scannerによる入力
* ファイル入出力
* 例外処理
* オブジェクト指向の基礎

---

# 3. クラス構成

## Todo

| フィールド     | 型       | 説明   |
| --------- | ------- | ---- |
| title     | String  | タスク名 |
| completed | boolean | 完了状態 |

### メソッド

| メソッド       | 説明        |
| ---------- | --------- |
| complete() | 完了状態にする   |
| toData()   | 保存用文字列を返す |

---

## TodoManager

### フィールド

```java
private ArrayList<Todo> todos;
```

### メソッド

| メソッド       | 説明    |
| ---------- | ----- |
| add()      | タスク追加 |
| show()     | 一覧表示  |
| complete() | 完了処理  |
| remove()   | 削除    |
| save()     | 保存    |
| load()     | 読込    |

---

## Main

メニューを表示し、ユーザー入力に応じてTodoManagerの各メソッドを呼び出す。

---

# 4. メニュー仕様

```text
========== TODO管理 ==========

1. タスク追加
2. 一覧表示
3. タスク完了
4. タスク削除
5. ファイル保存
6. ファイル読込
0. 終了

番号：
```

---

# 5. 機能仕様

## 5.1 タスク追加

### 入力

```text
タイトル：
```

### 処理

* Todoオブジェクトを生成
* ArrayListへ追加

### 出力

```text
登録しました。
```

---

## 5.2 一覧表示

### 表示例

```text
No 状態 タイトル

1 □ Java課題
2 ■ 掃除
3 □ レポート
```

状態

| 記号 | 意味  |
| -- | --- |
| □  | 未完了 |
| ■  | 完了  |

---

## 5.3 タスク完了

### 入力

```text
完了する番号：
```

### 処理

* completed を true にする

### 出力

```text
完了しました。
```

---

## 5.4 タスク削除

### 入力

```text
削除する番号：
```

### 処理

* 指定番号のタスクを削除

### 出力

```text
削除しました。
```

---

## 5.5 ファイル保存

保存ファイル

```
todo.txt
```

保存形式

```text
Java課題,false
掃除,true
レポート,false
```

保存完了

```text
保存しました。
```

---

## 5.6 ファイル読込

### 処理

* todo.txt を読み込む
* 1行ずつ split(",") で分割
* Todoオブジェクトへ変換
* ArrayListへ追加

### 成功

```text
読み込みました。
```

### ファイルが存在しない

```text
保存データがありません。
```

---

# 6. ファイル形式

1行につき1件

```
タイトル,完了状態
```

例

```text
数学,false
掃除,true
Java課題,false
```

---

# 7. エラー処理

## ファイルが存在しない

```text
保存データがありません。
```

---

## データ形式が不正

例

```text
Java課題
```

または

```text
Java課題,abc
```

表示

```text
読み込めないデータをスキップしました。
```

処理は続行すること。
